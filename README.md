## What's Inside

A dark-mode developer portfolio built with **Next.js 14**, **Tailwind CSS**, and **TypeScript** — statically exported and deployed to GitHub Pages.

### ✨ Features
- **About, Experience, Skills & Projects** — populated from my CV
- **Markdown Blog** — just add a `.md` file and push
- **SEO Optimized** — Open Graph, JSON-LD, sitemap, robots.txt
- **AI SEO** — `llms.txt` for ChatGPT, Perplexity & Google AI Overviews
- **Auto-Deploy** — GitHub Actions → `ahmednobii.github.io`

### 🛠 Tech Stack
| Layer | Tech |
|-------|------|
| Framework | Next.js 14 (App Router) |
| Styling | Tailwind CSS v3 |
| Blog | Markdown + gray-matter + remark |
| Deploy | GitHub Actions → GitHub Pages |
| Domain | [ahmednobii.me](https://ahmednobii.me) |

## Getting Started

```bash
npm install
npm run dev     # → http://localhost:3000
npm run build   # → static export to ./out
```

## Writing a Blog Post

Create a file in `content/blog/`:

```markdown
---
title: "Your Title"
date: "2026-05-09"
tags: ["android", "kmp"]
excerpt: "A brief summary."
---

Your content here...
```

Push to `main` → live within 1 minute.
